new preview(previewbox)

Use the API to parse a fragment of wikitext and render it as HTML.

The suggested implementation pattern (in Morebits.simpleWindow and Morebits.quickForm situations) is to construct a Morebits.wiki.preview object after rendering a Morebits.quickForm, and bind the object to an arbitrary property of the form (e.g. |previewer|). For an example, see twinklewarn.js.
